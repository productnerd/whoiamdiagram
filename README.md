# M's Life Dashboard

A retro-desktop home for my personal widgets. Themed like a classic Windows XP desktop, with each widget on its own tab.

**Live:** https://productnerd.github.io/mplifedashboard/

## Widgets
- **Who I Am** — an interactive "good life" map (the whoiamdiagram).
- **The Year** — an annual travel planner; paint weeks across locations, saved to your browser.

Single self-contained `index.html`, no build step. Deployed via GitHub Pages.
